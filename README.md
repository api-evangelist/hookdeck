# Hookdeck (hookdeck)
Hookdeck is a powerful tool that helps businesses streamline their webhook workflows. With Hookdeck, users can easily manage, monitor, and debug all of their webhooks in one centralized platform. By providing real-time notifications, detailed logs, and error handling capabilities, Hookdeck simplifies the process of integrating and testing webhooks. This innovative solution empowers businesses to ensure the reliability and efficiency of their webhook connections, ultimately improving their overall performance and productivity.

## APIs

### Hookdeck API
The Hookdeck REST API provides programmatic access to the Hookdeck Event Gateway platform. It enables management of sources, destinations, connections, transformations, events, attempts, metrics, and more. Authentication is via Bearer token or Basic auth using your project API key. The API uses date-based versioning in the base URL path.
